# react-boilerplate

## Includes
* webpack
* sass
* react
* babel

#### To start:
1. `git clone https://github.com/darkgraycat/react-boilerplate.git`
2. `npm install`

#### Development mode
`npm start`

#### Build application
`npm build`

Then just open `index.html` with Live Server


*node_modules folder size ~40 mb*
