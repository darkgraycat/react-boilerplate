# minimal react-boilerplate

## Includes
* webpack
* react
* babel

#### To start:
1. `git clone -b minimal https://github.com/darkgraycat/react-boilerplate.git`
2. `npm install`

#### Development mode
`npm start`

#### Build application
`npm build`

Then just open `index.html` with Live Server

*node_modules folder size ~25 mb*
